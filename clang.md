# C lang

## Differences between ' and " in C/C++

C言語でシングルクォートとダブルクォートは意味が違う． \
`'`は1つの文字(char)を表し，`"`は文字列(stringのリテラル)を表す．

`'a'`は1つの文字のリテラルを表し，`"a"`は1つの文字列リテラルを表す．つまり`"a"`は`'a'`と`null`を含んでいるので2つの要素を持つchar配列である．
